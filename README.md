# Kyblik
> Small Programming language which compiles into JavaScript
##### Work in progress!

## Installation
```bash
npm install -g kyblik
```

## Usage
```bash
# CLI help
kyblik --help

# Example
kyblik --file file.kyblik --output file.js
```
